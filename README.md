
# AI 软著文档智能生成系统

> 基于大语言模型的软件著作权申请材料自动生成工具，一键生成符合国家版权局规范的全套软著文档。

## 📋 功能概览

### 🏠 工作台
- 项目总览与统计面板
- 快速访问最近编辑的文档
- 项目状态一览

### 📁 项目管理
- 创建、编辑、删除软件项目
- AI 智能解析：输入一句话描述，自动提取项目名称、类型、行业、技术架构等信息
- 支持多项目管理，独立维护各项目文档

### 🤖 AI 智能文档生成
支持一键生成以下 7 类软著申请文档：

| 文档类型 | 说明 | 预计页数 |
|---------|------|---------|
| 软件功能说明书 | 详细描述软件各项功能、业务流程及界面交互 | 15-30页 |
| 软件设计说明书 | 包含系统架构、数据库设计、接口设计等技术细节 | 20-40页 |
| 用户操作手册 | 面向最终用户的系统安装、配置及使用指南 | 10-20页 |
| 软件简介 | 300字左右的软件概述，用于软著申请表填报 | 1页 |
| 技术实现说明 | 突出软件的技术创新点、核心算法及架构优势 | 3-5页 |
| 软件模块说明 | 按功能模块拆解的详细说明文档 | 10-15页 |
| 源代码说明 | 符合软著要求的前后30页代码结构说明 | 60页 |

**特色功能：**
- 流式输出，实时预览生成内容
- 支持导入已有文档作为基础内容，AI 在此基础上优化补全
- 支持 `.txt`、`.md`、`.html`、`.doc`、`.docx`、`.pdf` 格式导入
- 生成过程可随时**停止**，已生成内容自动保留
- 生成完成/失败均有明确提示，失败支持重试

### ✏️ 富文本编辑器
- 所见即所得的富文本编辑体验
- 支持加粗、斜体、删除线、链接、图片、分隔线等格式
- 左侧自动生成文档大纲，支持点击跳转
- 实时字数统计
- 自动保存

**AI 写作助手（右侧面板）：**
- **继续生成** — AI 断点续写，从末尾继续补全
- **内容扩写** — 增加细节与篇幅
- **内容精简** — 提取核心要点
- **专业化表达** — 转换为书面语
- **技术化描述** — 增加技术深度
- **逻辑优化** — 梳理上下文结构
- **生成流程图** — 自动生成可视化 Mermaid 流程图（渲染为图表，非代码）

**浮动 AI 工具栏：**
- 选中文本后自动弹出
- 快捷操作：扩写、精简、专业化、技术化
- 支持自定义 AI 指令输入
- 所有 AI 操作均支持**停止生成**

### 📦 导出中心
- **Word (.doc)** — 标准申报格式，可二次编辑，流程图自动转为图片
- **PDF** — 打印级排版，直接提交
- **Markdown** — 纯文本格式，适合开发者归档
- 按项目打包为 `.zip` 压缩包一键下载
- 自动处理 SVG 流程图转 PNG，确保 Word 中正常显示

### ⚙️ 系统设置
- AI 服务配置（API Key、模型选择、接口地址）
- 支持 DeepSeek 等大语言模型

### 🔐 登录系统
- 内置用户认证，保护系统访问
- 默认账号：`admin` / `admin123`
- 登录状态持久化（localStorage）
- 支持退出登录

## 🚀 快速开始

### 环境要求
- Node.js 16+
- 现代浏览器（Chrome、Edge、Firefox、Safari）

### 安装与运行

```bash
# 克隆项目
git clone <repository-url>
cd ai-softcopy-doc-system

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

### 配置 AI 服务

1. 登录系统（默认账号 `admin` / `admin123`）
2. 进入「系统设置」页面
3. 填入您的 DeepSeek API Key
4. 保存配置即可开始使用

## 📖 使用流程

```
创建项目 → 填写/AI解析项目信息 → 选择文档类型 → AI 生成文档 → 编辑优化 → 导出下载
```

1. **创建项目**：在「项目管理」中新建项目，输入软件名称等基本信息，或使用 AI 一句话智能解析
2. **生成文档**：进入「文档生成」，选择项目和需要的文档类型，点击「开始智能生成」
3. **编辑优化**：在「文档编辑」中对生成的文档进行人工审核和 AI 辅助优化
4. **导出下载**：在「导出中心」选择格式，一键打包下载

## 🛠️ 技术栈

- **前端框架**：React 18 + TypeScript
- **UI 样式**：Tailwind CSS
- **动画**：Framer Motion
- **本地存储**：IndexedDB（通过 idb 库）
- **AI 服务**：DeepSeek API（流式输出）
- **图表渲染**：Mermaid.js
- **文件处理**：JSZip（压缩导出）
- **图标**：Lucide React
